# foreversite

A hand-made esoteric static website designed to keep growing forever.

Rules of the place:

- Nothing should feel like a template.
- Every page should have its own local weather, voice, and visual law.
- Early web energy is good: tables, strange borders, tiny rituals, handmade navigation, odd files.
- Avoid filler. A small strange thing is better than a large bland thing.
- Internal links should feel like doors, emblems, loose floorboards, rumors, receipts, stains, and footnotes.
- CSS should vary wildly. Reuse shared CSS only for the site shell; individual pages should break away.

## Run

Open `index.html` directly, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Expansion

Read `FOREVER_NOTES.md` before adding new pages.
