# FOREVER NOTES

This is not a normal website. It is a cabinet that learned to hyperlink.

## Growth Pattern

Each expansion pass should add:

1. At least 3 new pages.
2. At least 3 different visual systems.
3. At least 1 non-obvious cross-link from an older page to a newer page.
4. At least 1 object/emblem/link that behaves like a secret door.
5. At least 1 page that is useful in some mundane way, but dressed strangely.
6. At least 1 new section path sometimes, e.g. `caution-zone/hallway2.html`, `redgarden/redgarden_home.html`, `transit/elevator-minus-one.html`, `municipal/tuesday-bucket.html`. Do not let the site collapse into only recipes, mazes, shrines, and objects.

## Anti-Staleness Rules

Do not repeat:

- The same page layout twice in a row.
- The same color palette twice in a row.
- The same narrator twice in a row.
- The same kind of mystery twice in a row.
- The same CSS trick twice in a row.

## Voice Rule

Write like a human who is specific, not like a model being whimsical.
Use concrete details: times, stains, prices, names of drawers, exact temperatures, ugly colors.

## File Naming

Prefer memorable paths:

- `recipes/cabbage-opera.html`
- `mazes/hedge-of-four-lefts.html`
- `objects/brass-cricket.html`
- `notices/laundry-moon.html`

## Long-Term Goal

Hundreds of pages. Hundreds of local CSS moods. A site that feels discovered, not produced.
